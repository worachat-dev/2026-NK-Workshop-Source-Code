# Enhancing M.3 Thai Science Education Through Interactive Digital Learning Games

This `README.md` file provides an overview of the project, its structure, and how to set up and use the interactive learning games.

---

## 🚀 Project Overview

This project, "Enhancing M.3 Thai Science Education Through Interactive Digital Learning Games," offers a collection of engaging, interactive web-based games designed to support Grade 9 (M.3) Thai science curriculum. The main goal is to make learning science concepts more fun and accessible through digital interaction.

The project is structured into two main parts, each containing several units, and within each unit, specific learning games. The central `main.html` acts as a portal, providing easy navigation to all available games.

## 🛠️ Setup and Usage

To access and play the games, follow these simple steps:

1.  **Clone or Download the Repository:**
    If you're using Git, open your Terminal (on macOS) or command prompt (on Windows) and run:
    ```bash
    git clone https://github.com/your-username/Enhancing-M3-Thai-Science-Education-Through-Interactive-Digital-Learning-Games.git
    ```
    (Replace `https://github.com/your-username/...` with the actual repository URL).

    Alternatively, you can download the project as a ZIP file and extract it to your desired location.

2.  **Navigate to the Project Directory:**
    Open your file explorer (Finder on macOS, File Explorer on Windows) and go to the root directory of the extracted project:
    `/Users/worachatwannawong/Downloads/Enhancing-M3-Thai-Science-Education-Through-Interactive-Digital-Learning-Games/` (or wherever you saved it).

3.  **Open `main.html`:**
    Locate the `main.html` file in the root of the project directory. Double-click this file. It will open in your default web browser.

4.  **Explore the Games:**
    The `main.html` page provides a user-friendly interface. You can click on any unit or game title to launch the respective interactive activity. Each game will open its `index.html` file within its specific sub-directory.

## ⚙️ Technical Details

* **Frontend Technologies:** The project is built using standard web technologies:
    * **HTML5:** For structuring the content.
    * **CSS3:** For styling the appearance and layout, including responsiveness.
    * **JavaScript:** (Presumed, for the interactive game logic within each `index.html` file, though not explicitly shown in `main.html`).
* **External Libraries:**
    * **Google Fonts:** Used for 'Poppins' (headings) and 'Inter' (body text) to enhance typography.
    * **Font Awesome:** Provides a wide range of scalable vector icons used throughout the `main.html` for better visual communication.
* **Design Principles:**
    * **User-Friendly Interface (UI):** Designed for intuitive navigation and a pleasant visual experience.
    * **Responsiveness:** The layout adapts to various screen sizes, from desktop monitors to mobile devices.
    * **Clear Hierarchy:** Content is organized with clear headings and visual cues to easily distinguish between parts, units, and individual games.
